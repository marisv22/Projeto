Cursos


✖

Design de jogos digitais :

Este curso abrange a criação e o desenvolvimento de jogos digitais, incluindo aspectos de narrativa, mecânicas de jogo, design de níveis, dentre outros.

▲

Programação de jogos :

Focado em linguagens de programação utilizadas no desenvolvimento de jogos, como C++, C#, Unity, Unreal, Engine, entre outras.

■

Arte e animação para jogos digitais :

Este curso aborda a criação de elementos visuais para jogos, incluindo modelagem 3D, animação, texturização e design de personagens.

◉

Áudio para jogos digitais :

Explora a criação e implementação de trilhas sonoras, efeitos sonoros e design de áudio para jogos.

Alguns Cursos

O curso de Jogos Digitais é uma graduação de nível superior com titulação tecnológica e duração média de dois anos para a formação. A graduação é para aquelas pessoas que têm o interesse de criar jogos de forma profissional a partir de um curso, e que consiga ingressar no mercado de trabalho. São alguns exemplos de cursos :

  • Design de Jogos Digitais;
  • Programação de Jogos;
  • Arte e animação para jogos digitais;
  • Áudio para jogos digitais.
